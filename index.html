A program that converts SimCity 2000 cities to Minecraft worlds. An online image editor with the look and feel of Microsoft Paint for Windows 95.
I have also prepared a few examples of working with images in Javascript: Image examples A 3D GUI similar to the UNIX seen in the movie Jurassic Park, done with built-in CSS transformations. In order to learn more about functional programming and specifically the Elm language, I decided to implement the game Connect-4. To see if I could make the implementation more optimal, I then implemented Connect-4 in Angular as well. And then I generalized the Minimax algorithm from Connect-4 and reused it to implement Checkers in the same project.
